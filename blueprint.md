
# Lotto Number Generator

## Overview

A simple web application to generate random lottery numbers.

## Features

- Generate 6 unique random numbers between 1 and 45.
- Display the generated numbers in a clear and visually appealing way.
- A button to generate a new set of numbers.

## Current Plan

1.  **`index.html`**: Set up the basic HTML structure with a title, a container for the numbers, and a button.
2.  **`style.css`**: Add styles for the layout, number display, and button to create a modern and clean look.
3.  **`main.js`**: Implement the JavaScript logic to generate and display the lottery numbers when the button is clicked.
